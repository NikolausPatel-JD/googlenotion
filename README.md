      - ## Overview
     
      - GoogleNotion bridges the gap between your search needs and knowledge management. Instead of manually copying search results to Notion, this tool automates the process, saving you time and keeping your research organized.
     
      - ## Getting Started
     
      - ### Prerequisites
     
      - - A Google account
        - - A Notion account
          - - API keys for both services
           
            - ### Installation
           
            - 1. Clone the repository:
              2.    ```bash
                       git clone https://github.com/NikolausPatel-JD/googlenotion.git
                       cd googlenotion
                       ```

                    2. Open `notion-google-search.html` in your web browser
                
                    3. 3. Configure your API keys when prompted
                      
                       4. ## Usage
                      
                       5. 1. Open the tool in your browser
                          2. 2. Enter your search query
                             3. 3. Click the search button
                                4. 4. Select results you want to save
                                   5. 5. Results will be automatically saved to your Notion workspace
